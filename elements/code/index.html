<code>

A code block is used to display computer code.

Overview

If you are not just talking about code but actually showing code, then use the code tag.

A <code> block is used to wrap blocks of computer code. Examples of HTML or other languages should be wrapped in code tags. Note that the act of wrapping code with code blocks will only change the appearance of the text. It will not escape HTML entities such as < or > or add line breaks.

Conflicting or Confusing Elements

<pre>

Many people make use of the <pre> block to wrap code since it handles line breaks automatically. Also the font used to render <pre> blocks looks a bit like computer code as well. But <pre> is meant for preformatted text and does not convey the same meaning as a block of code.
