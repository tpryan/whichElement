<q>

A q is a short, in-line quotation.

Overview

Use q correctly or don't use it at all, I always say.

A <q> is meant to be used to provide a short quote within some other content. The idea is to convey that it is a quote but not break out of the existing paragraph. Use q correctly or don't use it at all, I always say.

Conflicting or Confusing Elements

<blockquote>

So both <q> and <blockquote> denote quotes. The difference is that a <blockquote> is meant to do two things:

  1. Contain an extended quote
  2. Put a break in the content.
