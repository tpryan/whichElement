<dd>

The description part of a description list

Overview

Related Elements

  • <dl>
  • <dt>

A description list is for pairing one or more terms with one or more descriptions. The <dd> is the part that includes the description content. There can be multiple <dd> elements per <dt> element.

Another interesting observation from the spec is that you can have complex content in a <dd>. Lists, images diagrams, and tables are all valid under a <dd>.
