<ol>

Ordered list

Overview

Related Elements

  • <li>

A ordered list is is for displaying a sequential list of items. There is meaning to the order in which they appear in the markup and that ordinal count has meaning to the consumer of the info…

Um, in clearer language, this is a list. It displays numbers, or letters, but always in an order.

Example

  1. This item is first
  2. This item is second
  3. This item is third
