<p>

A p is a paragraph.

Overview

This tag is the basic building block of text on the web

A <p> is meant to group together multiple sentences into paragraphs. This tag is the basic building block of text on the web, and the first tag many people learned way back when they first cracked open the mysteries of HTML via view source.

A <p> is a block-level element that is not supposed to include other block-level elements. This makes it a poor candidate on which to base a complext concept, as <p> should be the basic element that you include in larger, more complex concepts.

Conflicting or Confusing Elements

<div>

So both <p> and <div> denote basic content. The difference is that <p> have meaning, specifically paragraph. A <div> is specifically meant to hold items and not have any special significance itself.
