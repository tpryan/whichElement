<dl>

Description list

Overview

Related Elements

  • <dt>
  • <dd>

A description list is for pairing one or more terms with one or more descriptions. It's important to note that there can be a many to many relationship there. You can link multiple descriptions to one term, or multiple terms to one description. These are commonly used for sets of definitions, and for F.A.Q.'s.

It's important to note that previous to this, <dl> elements were definition lists and this brought semantic arguments as to how they could be used or not used. The W3C appear to have taken this issue head on by changing the spec.
