<li>

List item

Overview

Related Elements

  • <menu>
  • <ol>
  • <ul>

A <li> is the appropriate child of one of several types of HTML lists.

Examples

An unordered list (<ul>)

  • This item is first, but I don't care.
  • This item is second, but I don't care.
  • This item is third, but I don't care.

An ordered list (<ol>)

  1. This item is first
  2. This item is second
  3. This item is third
