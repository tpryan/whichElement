<dt>

The term being decribed by a description list

Overview

Related Elements

  • <dl>
  • <dd>

A description list is for pairing one or more terms with one or more descriptions. The <dt> is the part that includes the term being described. There can be multiple <dt> elements per <dd> element.

Another interesting observation from the spec is that you can have complex content in a <dt>. Lists, images diagrams, and tables are all valid under a <dt>.
