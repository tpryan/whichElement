<details>

Content that is intended to be hidden away, and revealed at user request

Overview

Related Elements

  • <summary>

You ever do that thing with the JavaScript where you hide content, only to reveal it when the user clicks on a control? Yeah, this is that. But with some HTML and no JavaScript. Cool huh? Sad thing is that it only works in Webkit browsers for now.
