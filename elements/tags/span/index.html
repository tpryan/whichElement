<span>

An inline level element with no semantic meaning at all.

Overview

The job of the <span> element is to provide hooks for other attirbuted to be applied to content. In mosts cases, it is a markup construct that allows you to target content with a CSS class.

Conflicting or Confusing Elements

<div>

A <div> is a generic block element, not inline.
