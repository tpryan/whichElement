<menu>

A list of commands usually as part of a web application.

Overview

Related Elements

  • <li>

The <menu> is intended to be part of web applications. It allows for the creation of either context menus or toolbars.

Conflicting or Confusing Elements

<nav>

The easiest way to keep these two clear in your head is:

  • The <nav> element is for navigating through links on your site.
  • The <menu> element for giving commands like you would ordering off of a menu at a restaurant… That's terrible. You know what? Just remember the one for <nav>
