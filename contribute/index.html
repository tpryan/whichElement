Contribute

There are a number of ways in which you can contribute to whichElement.com:

Good

Drop a line with an issue you have with something on the site.

Better

File a bug report through github.

Best

Fork the code, fix the issue, send it back to us.

whichElement is meant to be a group effort. Please pitch in.
